# 真随机数生成器

Hi！ 这是个开源的易语言模块，它可以生成真随机数。
代码来源于 https://www.cnblogs.com/mchmch/p/generate-real-random-number-in-c-plus-plus.html ，我仅仅只是将其翻译为易语言。

它仅仅只是我用来练习易语言的模块，如有问题请提 Issue！
欢迎 Issue 和 Pull request！

## 使用
### 模块
可以从 Release 页面直接获取，也可以编译源码。
#### 从 Releases 页面获取
单击网页右方的 `[Releases](https://github.com/Haceau-Zoac/ERandomModule/releases)`，然后单击最上方一个版本下面的 `真随机数生成器模块....ec` 。
#### 编译
##### 需求
* Git（非必须）
* 易语言
##### 获取代码
若有 Git，则可使用命令 `https://github.com/Haceau-Zoac/ERandomModule.git`。
若速度过慢，可使用 GitHub 镜像 `https://github.com.cnpmjs.org/Haceau-Zoac/ERandomModule.git`。
或者下载代码，单击网页上方的 `Code` 按钮，然后单击 `Download ZIP`，并解压。
##### 编译
使用易语言打开 `src/真随机数生成器.e`，然后点击 `编译->编译` ，或单击 `F7`。
